# PRODIGY_WD_01
This is a simple and responsive honey website built using HTML, CSS, and JavaScript. The website showcases honey products, a hero section with a beehive theme, product images, and basic contact information. 
The website includes an interactive fixed navigation bar that changes style on scroll, smooth scrolling between sections, hover effects on images and buttons, and a mobile-friendly hamburger menu for smaller screens.
Tech Stack:
HTML5 – Structure
CSS3 – Styling, Flexbox/Grid, Animations
JavaScript (ES6) – Interactivity & Scroll Effects
